composer create-project laravel/laravel example-app > Install laravel

php artisan migrate > create database tables

php artisan migrate:rollback > remove database tables

php artisan make:model -cfm Controllername > create (model, controller, migrationTable, factory)

php artisan make --help > get make help

php artisan db:seed > Create faker data in database table

php artisan migrate:refresh --seed > Re-create the faker data in database table

php artisan tinker > Build mysql quries

composer require laravel/ui > install bootstrap
php artisan ui bootstrap --auth > configure bootstrap ui and laravel auth
npm install > install npm
npm run dev > configure npm modules
npm run watch > run watch for CSS and other changes

php artisan db:seed > run default database table seed
